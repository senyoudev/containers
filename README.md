# Chapter 2: Namespaces
 
 ## Overview
This chapter isolates the process PID.

 ## Usage

 ### Using Golang
 ```sh
 go run main.go run echo "Hello from container"
 ```

 ### Using Docker
 ```sh
 docker run <image_name> run echo "Hello from container"
 ```

 ### Using OContainer
 ```sh
 OContainer run echo "Hello from container"
 ```
