# Chapter 2: Namespaces
 
 ## Overview
This chapter introduces **Linux namespaces**, starting with UTS (hostname) isolation.

 ## Usage

 ### Using Golang
 ```sh
 go run main.go run hostname
 ```

 ### Using Docker
 ```sh
 docker run <image_name> run hostname
 ```

 ### Using OContainer
 ```sh
 OContainer run hostname
 ```
